===================
:sortable:
===================

Adding the ``:sortable:`` field enables table sorting.

Sorting cycle:

* :menuselection:`Original Sort Order --> Alphanumeric --> Reverse Alphanumeric --> Original Sort Order`

.. rcsv-table::
    :header-rows: 2
    :sortable:

    People, <
    Name, Age 
    Tom, 22
    Alisa, 44
    Mary, 33
    John, 66
    Nate, 77
    Janice, 55
