Sum#

Totals numeric values from cells, ranges, or literal arguments. If no numeric values are found, SUM() returns 0.

Function#

  • =SUM(VALUE; VALUE; ...)

  • =SUM(RANGE)

Function VALUE Field#

Each VALUE can be:

  • A literal number

  • A cell reference

  • A range reference

  • Another formula that renders a numeric value

Blank values are ignored.

Non-numeric values are ignored and produce a warning.
